# Mortgage Calculator AI MCP

Calculate mortgage payments, amortization schedules, and total interest for home loans.

## Tools

- `calculate_monthly_payment` — Calculate fixed monthly mortgage payment.
- `generate_amortization_schedule` — Generate year-by-year amortization summary.
- `compare_mortgages` — Compare two mortgage scenarios side-by-side.

## Install

```bash
pip install -e .
```

## Run

```bash
python -m mortgage-calculator-ai-mcp.server
```
